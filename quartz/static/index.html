<p><strong>Run in Terminal macos</strong></p>

<ul>

<li>

<p>Install <em>libreoffice</em></p>

<pre><code> docker run -d \

 --name=libreoffice \

 --security-opt seccomp=unconfined \

 -e PUID=1000 \

 -e PGID=1000 \

 -e TZ=Etc/UTC \

 -p 3005:3000 \

 -p 3006:3001 \

 --restart unless-stopped \

 lscr.io/linuxserver/libreoffice:latest

</code></pre>

</li>

<li>

<p>Install <em>Obsidian</em></p>

<pre><code>  docker run -d \

  --name=obsidian

  -security-opt seccomp=unconfined \

  -e PUID=1000 \

  -e PGID=1000 \

  -e TZ=Etc/UTC \

  -p 3003:3000 \

  -p 3004:3001 \

  --shm-size=&quot;1gb&quot; \

  --restart unless-stopped \

  lscr.io/linuxserver/obsidian:latest

</code></pre>

</li>

</ul>
